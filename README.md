> 上周答应帮一个远在韩国的读研同学写一个测试系统，要求测试人员回答两个问题，每题回答时间15分钟，有倒计时

### 两种测试类型

* 第一题必须答完15分钟，然后自动解锁第二题，关闭第一题，不可回到第一题
* 两题可以自动切换回答，每题解答时间15分钟，半个小时候自动提交并生成一个编号
